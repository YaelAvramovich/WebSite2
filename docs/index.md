## About this site

This site was built using **MkDocs** with the **Material** theme. Key features include:

- 🎨 **Gradient colour theme** throughout the design
- 📱 **Fully responsive** — looks great on any device
- ♿ **Accessible** — semantic HTML, ARIA labels, and high-contrast focus indicators
- 🌗 **Light / Dark mode** — automatically detects your OS preference, switchable any time
- ✍️ **Blog** — a dedicated section for posts, categorised and archived automatically
- 📬 **Contact** — a clean contact form to get in touch

Feel free to explore the site using the navigation bar above, or jump straight to the [blog](blog/) or [contact](contact/) page.
